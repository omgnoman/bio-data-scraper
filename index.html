What We Do

We get papers from NCBI.

We save research papers in our own database and present you in a way that you can access them in a number of ways.

Get Data from NCBI

We download data from NCBI in XML file. For this purpose we must choose a Specific field of study and get it's related papers in form of XML file.

Load in Private Database

We have build a python script which loads the papers data into SQLite. The script made database and creates tables in the database as we instruct it.

Presenting Data

The NCBI research paper will be shown on HTML/CSS page. On the server side php is working which will brought data from database according to the query we entered.

Data Sources

We get data from number of website for your ease. But for now, we are using only NCBI and UNIPROT as our primary Data Source.
